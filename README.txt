#
Para compilar este programa, simplesmente utilize o comando 'make':
> make [-f Makefile]

Um arquivo binario (executavel) "matrix" sera gerado no diretorio corrente.
Para utilizar o programa basta executar o binario,
> ./matrix

Opcionalmente voce pode gravar as matrizes de entrada num arquivo texto com estrutura igual aos exemplos "input1.dat" e "input2.dat" presentes no diretorio.
> ./matrix < meu_input.txt
